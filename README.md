# Hero-with-Blueprint-Animation
Description
Complete hero section featuring:

Left column: Headline, subheadline, benefits, CTAs
Right column: Animated SVG blueprint (Strategy → Data → Teams → Compliance)
Ultranoir-inspired visual style with Florian Nègre branding

Branding

Primary Color: #0d3c70 (Navy Blue)
Accent Color: #d6ff3f (Lime Yellow)
Background: linear-gradient(135deg, #0a1628 0%, #0d3c70 100%)

Features

Pure HTML/CSS/SVG (no dependencies)
Responsive design (desktop/tablet/mobile)
Sequential animations (boxes appear one by one)
Hover effects on blueprint boxes
Pulse dots animating through connections
~12KB total weight

Usage

Copy hero-ultranoir-style.html content
Paste into UMSO Custom Code section
Adjust CTA links as needed
Deploy
